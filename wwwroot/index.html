About

The LAN Scanner Tool is a powerful network scanning utility that helps you discover devices on your local network. It uses Nmap for efficient host discovery and tries multiple methods to resolve hostnames.

Features:

  • Scan entire subnets for active hosts
  • Resolve hostnames and FQDNs when available
  • Export results to CSV format
  • Web interface for easy access

You can also use this tool from the command line: 

dotnet run 192.168.1
